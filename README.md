# Stopwatch v1 #

#### Create a new instance ####
`var timer = new window.stopwatch();`

#### Start the timer ####
`timer.start();`

#### Pause the timer ####
`timer.pause();`

#### Restart the timer when paused ####
`timer.start();`

#### Reset the timer ####
`timer.reset();`

#### Get the formatted time ####
`var elapsed = timer.current('mm:ss');`
